# AI Quant Trading Strategy 📈

This project implements a machine learning-based trading strategy using XGBoost.

## Features
- Technical indicators (MA, RSI, Volatility)
- Probability-based trading signals
- Risk management (loss clipping)
- Backtesting vs market performance

## Tech Stack
- Python
- Pandas, NumPy
- XGBoost
- Matplotlib

## Results
The strategy was evaluated using cumulative returns and Sharpe Ratio.
